## basic scatter plot

- suppose we have data that looks like,

```
   species island    bill_length_mm bill_depth_mm flipper_length_mm body_mass_g sex     year
   <fct>   <fct>              <dbl>         <dbl>             <int>       <int> <fct>  <int>
 1 Adelie  Torgersen           39.1          18.7               181        3750 male    2007
 2 Adelie  Torgersen           39.5          17.4               186        3800 female  2007
 3 Adelie  Torgersen           40.3          18                 195        3250 female  2007
 4 Adelie  Torgersen           NA            NA                  NA          NA NA      2007
 5 Adelie  Torgersen           36.7          19.3               193        3450 female  2007
 6 Adelie  Torgersen           39.3          20.6               190        3650 male    2007
 7 Adelie  Torgersen           38.9          17.8               181        3625 female  2007
 8 Adelie  Torgersen           39.2          19.6               195        4675 male    2007
 9 Adelie  Torgersen           34.1          18.1               193        3475 NA      2007
10 Adelie  Torgersen           42            20.2               190        4250 NA      2007
```

- To draw scatter plot, where bill_length_mm goes to x axis and bill_depth_mm goes to y axis, use the code below.

```R
penguins %>% ggplot(aes(x=bill_length_mm, y=bill_depth_mm)) + geom_point()
```

## Adding color information using fill

- Use the code below to add color to dots.

```R
penguins %>% ggplot(aes(x=bill_length_mm, y=bill_depth_mm, color=species)) + geom_point()
```

- The result looks like,

![Image](/static/images/r/scatter_color.png)

## Split the graph using facet_wrap

- If you want to have multiple graphs, you can use facet_wrap.
- The code below produces two graphs with female on left, male on right.

```R
penguins %>% ggplot(aes(x=bill_length_mm, y=bill_depth_mm, color=species)) + geom_point() + facet_wrap(vars(sex))
```
