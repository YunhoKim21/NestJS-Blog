회사 일을 하기 위해 어느날 ssh로 회사 서버에 접속하였더니 다음과 같은 오류가 발생하였다. (뭔가 굉장히 무섭게 생겼다.)

![image](/static/images/fig.png)

시키는 대로 나의 system administrator 에게 연락하였더니, 다음과 같이 해결해 주셨다.

1. .ssh/known_host (로컬) 파일 열기

```
vim ~/.ssh/known_host
```

2. 내부에 있는 데이터 중 접속하고자 하는 ip 로 시작하는 줄 모두 지우기.

3. terminal 종료 후 다시 실행, ssh 접속

하였더니 해결!
